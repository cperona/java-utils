# Java Utils

Program with tools to manage files and directories, and read and write information on the file system.

## Exercise 1
Create a class that lists the contents of a directory received as a parameter in alphabetical order.

## Exercise 2
Extend the previous class to recursively list a directory tree, displaying the contents of all levels. The results must be shown in alphabetical order within each level, indicating whether each item is a directory (D) or a file (F), along with its last modification date.

## Exercise 3
Modify the previous exercise so that, instead of displaying the output on the screen, the result is saved into a TXT file.

## Exercise 4
Add functionality to read any TXT file and display its contents in the console.

## Exercise 5
The program must serialize a Java object into a .ser file and then deserialize it.
